            # 踩坑汇总

            # ES6的兼容性

            • 问题描述：安卓5、6部分手机的浏览器版本不支持ES6语法的箭头函数等写法。使用ES6的相关语法会报错。

            # data()方法的问题

            • data()方法获取DOM元素中的data-自定义属性会有缓存，只能获取到第一次data-的自定义值，若发生改变，无法获取。

            # 微信中reload兼容性问题

            • 微信环境或者某些浏览器中存在使用reload,即使设置了reload(true)也无法清缓存刷新，要刷新尽量使用location.href +时间戳进行刷新。

            # vivo部分手机的webview刷新机制

            • vivo部分手机webview的刷新机制为：关闭当前窗口，新建一个窗口加载当前页面达到刷新效果，所以会导致sessionStorage存储失效。

            # 微信小程序真机调试

            • 微信小程序在真机调试模式下，如果request请求数据过大（包含图片实例之类的）的情况下，会假死，与程序无关，发布到体验版即解决bug。
