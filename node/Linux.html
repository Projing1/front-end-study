# Linux快速上手

# Linux简介

  • Linux内核最初只是由芬兰人林纳斯.托瓦茨在赫尔辛基大学上学时出于个人爱好而编写的。
  • Linux是一套免费使用和自有传播的类Unix操作系统，是一个基于POSIX和UNIX的多用户，多任务，支持多线程和多CPU的操作系统。
  • Linux能运行主要的UNIX工具软件，应用程序和网络协议，它支持32位和64位硬件，LInux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统；

# Linux应用领域

  • 服务器系统：以Linux为基础的LAMP技术，是网站服务供应商最常使用的平台；
  • 桌面系统： 新版本的Linux系统特别是在桌面应用方面进行了改进，完全可以作为图形界面操作系统来使用。
  • 嵌入式系统： 现在很多移动设备也采用基于Linux的嵌入式系统，例如机顶盒、移动电话和移动设备等；
  • 电子政务： Linux在安全性方面的独特优势，使得他在政府应用领域大行其道，目前一些国家正在将其电子政务系统向Linux平台迁移；

# 查看IP

  • ifconfig ：mini版没有安装此命令，需要自己安装。
  • ip addr ：显示IP信息

# yum 命令

  • yum 是一个在CentOS中的shell前端软件包管理器。
  • yum update 更新所有软件；
  • yum install 安装指定软件；
  • yum remove 删除指定软件。

# 系统信息

  • free -m: 内存信息；df -h：硬盘信息; cat/proc/cpuinfo :CPU信息 ;

# 文件系统

  • cd:进入目录； ls：列出目录信息，ls -al（ll）:列出详细信息 touch新建文件 mkdir新建目录 rm:删除文件或目录； cp:复制； mv: 移动（重命名）； |通过管道传递命令；

# 压缩及解压

  • tar:用于压缩及解压； tar -czvf:打包压缩； tar -tzvf:列出压缩文件的内容 ； tar -xzvf: 解压文件 ;

# 用户操作

  • useradd/adduser:添加用户； password:设置密码； userdel:删除用户 ； sudo: 提权 ;
